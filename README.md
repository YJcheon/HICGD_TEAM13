# StoryLine
때는 20XX년, 외계인의 침공으로 인해 인류는 멸망하였다.<br>
인류의 마지막 군인인 ‘천조교’는 외계인들의 핵심시설인 ‘아마겟돈’에 잠입하여 그들의 생명유지장치인 ‘하이퍼코어’를 파괴해야 한다.<br>
하지만 그곳은…… 끊임없는 미로였다……<br>
미로 어딘가에 위치한 ‘하이퍼코어’를 찾아라!
